# 🍔 Food Delivery Web Application

## 📌 Project Overview

The Food Delivery Web Application is a responsive front-end website developed using **HTML, CSS, and JavaScript**. It provides an intuitive interface for users to browse food items, explore restaurant menus, and place orders through a modern and user-friendly design.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices.
* Attractive and modern user interface.
* Browse food items by category.
* Search functionality for quick food discovery.
* Add food items to the cart.
* Dynamic cart management using JavaScript.
* Interactive navigation and smooth user experience.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.
